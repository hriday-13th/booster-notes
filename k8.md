## Apply a file
kubectl apply -f <file>

## Namespaces
What it is: Logical isolation boundary in the cluster
What it does: Groups resources, scopes names, enables RBAC and quota separation

kubectl get pods -n my-namespace

kubectl config set-context --current --namespace=my-namespace

## ConfigMap
What it is: Key-value config store for runtime settings
What it does: Supplies dynamic config
